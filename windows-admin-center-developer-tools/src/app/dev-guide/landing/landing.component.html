Welcome to Windows Admin Center Developer Tools! Yaaaas

In this toolset, you will find development examples and scenarios to help you build an extension in Windows Admin Center. With these examples, you can interact with them in runtime, as well explore the included source code in the SDK. Be sure to check out documentation included in the SDK, which has step-by-step instructions and additional information to help you get an extension up and running in Windows Admin Center.

Developer Tools include the following sections:

  • Introduction
  • Development Guide
  • IFrame Example
  • Solution Example

Introduction

The following API examples are included in the Introduction tool. Click on one for more information.
  • WMI
  • PowerShell
  • Tree View Code
  • User Profile
  • Notifications
  • Gateway Extensions

Development Guide

The Development Guide tool provides an in-depth look at all the individual controls that are available in Windows Admin Center, and you can find examples of how to use each control in the source code, included in the SDK.

Additional Examples

Be sure to check out these additional examples, available in the SDK.
  • Extension Template
    • Start a new extension from a blank slate. Includes step-by-step examples for how to make modifications to the included code, build, and side load your extension in Windows Admin Center.
  • IFrame Example
    • See how to implement an IFrame in an extension in Windows Admin Center.
  • Solution Example
    • See how to implement a Solution in Windows Admin Center.
